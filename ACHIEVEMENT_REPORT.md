## 🎯 核心功能演示

### 想法澄清流程

```
用户输入模糊想法 → ClarifierAgent分析 → AI生成澄清问题 → 用户回答 → 结构化想法输出
```

### 创新优化流程

```
结构化想法 → InnovatorAgent分析 → 6维度创新建议 → 评分排序 → 最优建议输出
```

### AI 模型选择

```
Agent请求 → ModelManager路由 → 自动选择提供商 → API调用 → 智能响应处理
```
